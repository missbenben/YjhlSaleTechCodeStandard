## 自动化工具
- 定义
    为了减少重复性工作，在可以自动化的前提下编写的支持部门工作的前端页面
- 自动化工具的编写要使用React，每个组件的编写要考虑到复用性
- 自动化工具应该具有版本管理的意识，并且每次版本升级要写版本升级的点
